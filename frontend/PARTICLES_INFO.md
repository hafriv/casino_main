# Particles.js Integration

## Описание
На сайт добавлен анимированный фон с частицами (particles.js) для создания современного и динамичного визуального эффекта.

## Установленные пакеты
- `tsparticles` - основная библиотека
- `@tsparticles/react` - React-обертка
- `@tsparticles/engine` - движок частиц
- `@tsparticles/slim` - облегченная версия для лучшей производительности

## Компонент ParticlesBackground

Расположение: `frontend/src/components/ParticlesBackground.jsx`

### Особенности конфигурации:
- **Количество частиц**: 60
- **Скорость**: 1 (медленная, не отвлекает)
- **Прозрачность**: 0.3 (ненавязчивая)
- **Связи между частицами**: включены с прозрачностью 0.15
- **Интерактивность**: отключена (не реагирует на мышь)
- **Позиционирование**: `position: fixed, z-index: -1` (всегда на заднем плане)
- **pointer-events: none** - не мешает взаимодействию с элементами

## Где используется:
1. **MainPage** - показывается после окончания вступительного видео
2. **LuckyJet** - на странице перенаправления
3. **Games** - на странице "Игры скоро появятся"
4. **Roulette** - на странице рулетки

## Настройка

Для изменения внешнего вида частиц отредактируйте файл `ParticlesBackground.jsx`:

- **Цвет**: измените `particles.color.value`
- **Количество**: измените `particles.number.value`
- **Скорость**: измените `particles.move.speed`
- **Прозрачность**: измените `particles.opacity.value`
- **Связи**: настройте `particles.links`

## Производительность

Компонент использует облегченную версию библиотеки (`@tsparticles/slim`) для минимального влияния на производительность. Частицы рендерятся с использованием Canvas API и оптимизированы для плавной работы даже на слабых устройствах.
